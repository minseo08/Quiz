# Quiz 웹 애플리케이션

## 개요

-spring 기초를 다지기 위한 나만의 실습
-이 프로젝트는 다양한 유형의 퀴즈(선택형 퀴즈, 단답형 퀴즈, 이미지 기반 퀴즈)를 제공하는 웹 애플리케이션. 
-사용자는 퀴즈를 풀고 결과를 확인할 수 있으며, 관리자는 새로운 퀴즈를 추가할 수 있는 기능.
-관리자가 추가한 퀴즈는 H2 database에 저장할 수 있게 연동(JPA)

## 기술 스택

- **프레임워크**: Spring Boot
- **템플릿 엔진**: Thymeleaf
- **데이터베이스**: H2 (테스트 환경), JPA 사용
- **빌드 도구**: Gradle
- **프론트엔드**: HTML, CSS
